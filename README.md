# Advanced AI Tic Tac Toe Game

This project was developed as part of the SkillCraft Technology Web Development Internship.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript

## ✨ Features

- Player vs Player Mode
- Player vs AI Mode
- Winner Detection
- Draw Detection
- Scoreboard System
- Restart & New Game Options
- Glassmorphism Neon UI
- Responsive Design
- Interactive Animations

## 🎮 Game Modes

### Player vs Player
Two players can play against each other.

### Player vs AI
Play against computer-generated moves.

## 📂 Project Structure

- index.html
- style.css
- script.js

## 🎯 Objective

To build an advanced interactive Tic Tac Toe game using JavaScript with AI logic, responsive design, and modern UI features.
